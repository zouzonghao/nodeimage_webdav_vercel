通过 GitHub Actions 触发同步

在此处输入您的 GitHub 信息以手动触发同步工作流。所有凭据都应作为 Secrets 存储在您的 GitHub 仓库中。

首次配置指南

请在您的 GitHub 仓库的 Settings > Secrets and variables > Actions 页面中，添加以下 Secrets：

  • 必需: NODEIMAGE_COOKIE, WEBDAV_USERNAME, WEBDAV_PASSWORD, WEBDAV_BASE_PATH
  • 可选 (有默认值): NODEIMAGE_API_URL, WEBDAV_URL, HTTP_PROXY_URL
